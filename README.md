zenpencils
=========


A mini scraper to grab all the posters from zenpencils [website]


Version
----

1.0


Requirements
----

- Node (>= v0.11.13)
- Good Internet Connection


Installation
--------------

```sh
git clone https://github.com/jigarjain/zenpencils zenpencils
cd zenpencils
npm install
mkdir -p posters
```


Run
--------------

```sh
node --harmony index.js
```
